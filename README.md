# 📊 Bank Lending Intelligence Dashboard | Power BI

## 🔎 Project Overview
Developed an interactive **Bank Lending Intelligence Dashboard** in Power BI to analyze loan applications, funded amounts, repayments, loan quality (Good vs. Bad), customer demographics, and key lending KPIs.  
This project demonstrates strong skills in **data modeling, DAX, Power Query, and SQL integration**.

---

## 📂 Repository Contents
- **Bank Lending Intelligence Dashboard.pbix** → Power BI dashboard file  
- **financial_loan.csv** → Dataset used for analysis  
- **Sql queries Of Bank Lending Intelligence Dashboard** → SQL queries used for data extraction  
- **Screenshots/** → Dashboard visuals (Summary, Overview, Details, Model View, Published)  
- **README.md** → Documentation  
- **LICENSE** → MIT License
